DMCian App

Overview

The Android DMCian app was developed as my BCA Final Year Project. It helps students connect with the college counselor, receive announcements, and manage appointments efficiently.

Features

- Appointment scheduling with counselors.
- Department-specific notices.
- User-friendly interface with secure login.
- Push notifications for announcements.

Technologies Used

- Frontend: Java (Android Studio)
- Backend: Firebase
- Tools: XML for UI, GitHub for version control
